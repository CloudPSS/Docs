### 下载与安装

CloudPSS可以直接从这里[PyPi_](https://pypi.org/project/cloudpss/)下载, 但是更方便的方式是通过`pip_`，直接运行下面的命令进行`安装`:
```[pyhton]
pip install cloudpss
```
需要`升级`的话，执行下面的命令:
```[pyhton]
pip install --upgrade cloudpss
```
### 完全卸载

执行下面的命令进行`完全卸载`:
```[pyhton]
pip uninstall cloudpss
```

**CloudPSS IESlab** 提供了丰富的综合能源系统元件，其分类及具体元件组成如下表所示：

| 分类 | 元件 |
| :--- | :---  |
| 电源元件 | 光伏系统、风机、燃气轮机、外部电源 |
| 冷热源元件 | 热泵、燃气锅炉、热管式太阳能集热器、电压缩制冷机、吸收式制冷机 |
| 储能元件 | 蓄电池、储水罐 |
| 输配电系统传输元件 | 变压器、传输线、模块化多电流变压器MMC |
| 供热系统传输元件 | 离心泵、管道 |
| 负荷元件 | 电负荷、采暖制冷负荷、建筑物负荷 |

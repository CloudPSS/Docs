## 基本描述

> **该元件指热管式太阳能集热器的设备设施;$A_{p}$是集热器的吸热板面积，I是太阳辐射强度，$U_{L}$是总热损系数(W/m2·℃)，$T_{p}$是集热板平均温度，$T_{a}$是环境温度,集热器得到的有用能量$Q_{u}$为：**
> $$Q_{u} = A_{p}I - A_{p}U_{L}\left( {T_{p} - T_{a} } \right)$$

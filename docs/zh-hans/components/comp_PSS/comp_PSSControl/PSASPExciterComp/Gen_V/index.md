## 使用说明
发电机机端电压变送器框图如下所示。

![等效图](./GenV.png)

模型可以选择两种计算方式：
>  + Input 3 phase vectors: 输入电压电流三相瞬时值，利用Clark变换计算Vc
>  + Input PQ: 输入电压电流有效值，以及额外的有功无功以计算功率因数

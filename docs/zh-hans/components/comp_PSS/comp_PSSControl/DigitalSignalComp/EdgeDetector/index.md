## 基本描述


> **该元件实现对输入信号的跳变检测。**

## 参数列表
### Configuration
| 参数名 | 单位 | 备注 | 类型 | 描述 |
| :--- | :--- | :--- | :--: | :--- |
| Name |  | 元件名称 | 文本 | 此处输入边缘检测器的名称（可缺省） |
| Output for Positive Transition |  | 上升沿输出值 | 实数（常量） | 当检测到输入上升沿的输出值 |
| Output for No Transition |  | 无变化输出值 | 实数（常量） | 当检测到输入无变化时的输出值 |
| Output for Negative Transition |  | 下降沿输出值 | 实数（常量） | 当检测到输入下降沿的输出值 |

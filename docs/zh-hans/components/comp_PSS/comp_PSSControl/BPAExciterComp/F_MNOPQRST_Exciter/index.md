## 基本描述
> **该元件用以建模励磁器，可配置为以下8种BPA励磁器：**
>  + FM型励磁器
>  + FN型励磁器
>  + FO型励磁器
>  + FP型励磁器
>  + FQ型励磁器
>  + FR型励磁器
>  + FS型励磁器
>  + FT型励磁器

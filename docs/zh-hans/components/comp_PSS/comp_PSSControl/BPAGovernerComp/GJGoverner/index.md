| 参数名 | 单位 | 备注 | 类型 | 描述 |

| ε | p.u. | 转速偏差死区ε（相对系统频率的标么值，死区为±0.5ε） | 实数（常量） |  |

| Kd | p.u. | PID微分环节倍数KD | 实数（常量） |  |

| INTG_MIN | p.u. | PID积分环节限幅下限INTG_MIN | 实数（常量） |  |

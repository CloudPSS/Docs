### 下载与安装

CloudPSS可以直接从这里PyPi_下载, 但是更方便的方式是通过pip_ ，直接运行下面的命令进行安装: 
```[pyhton]
pip install cloudpss
```
需要升级的话，执行下面的命令:
```[pyhton]
pip install --upgrade cloudpss
```
### 完全卸载

执行下面的命令进行完全卸载:
```[pyhton]
pip uninstall cloudpss
```

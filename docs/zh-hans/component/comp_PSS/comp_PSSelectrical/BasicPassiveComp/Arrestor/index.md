## 使用说明
避雷器在正常工作电压下其电阻值很高，实际上相当于一个绝缘体，而在过电压作用下，电阻片的电阻很小，残压很低。CloudPSS的避雷器采用分段线性电阻串联可变电压源进行模拟。下图示出了避雷器的伏安特性曲线及其分段线性化过程。

![曲线图](./曲线图.png)

根据该曲线，可计算出避雷器的等效电路，如下图所示。

![等效图](./等效图.png)

计算公式如下：
$$v = E + Ri$$
$$E = \frac{ {x_2}{y_1} - {y_2}{x_1} }{ {x_2}-{x_1} }$$
$$R = \frac{ { {y_2} - {y_1} } }{ { {x_2} - {x_1} } }$$

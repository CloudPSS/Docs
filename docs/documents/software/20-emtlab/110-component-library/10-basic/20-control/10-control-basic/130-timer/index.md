### 使用说明
**计时器**用于对高电平或者低电平输入信号进行计时，输出为计时时间。用户可通过选择**外部控制**或者**内部控制**方式来控制清零模式，外部控制方式下，清零/保持由外部输入引脚决定，内部控制方式下，清零/保持由用户在参数配置时选择。

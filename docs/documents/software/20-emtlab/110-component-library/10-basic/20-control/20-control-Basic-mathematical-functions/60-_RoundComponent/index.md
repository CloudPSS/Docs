## 元件定义
该元件实现对输入信号进行取整，可选择 `Round`、`Ceil` 或 `Floor` 取整方法：
- `Round` 方法：四舍五入。
- `Ceil` 方法：向上取整。
- `Floor` 方法：向下取整。

## 元件定义
该元件实现指数运算，可配置底数为 10、自然常数 e 或自定义常数 a。函数式为：$y(t)=10^{u(t)}$ 或 $y(t)=e^{u(t)}$ 或 $y(t)=a^{u(t)}$, $u(t)>0$。

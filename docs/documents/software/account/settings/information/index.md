点击页面左下角的账户**设置**按钮，进入的默认页面就是**个人信息设置**页面。该页面显示内容如下：

+ **账号名称**：账号名称唯一不可编辑修改。

+ **电子邮箱/手机号**：本页面不可编辑修改，修改参见[账户安全设置](../security/index.md)页面内容。

+ **昵称**：可编辑(必填)。

+ **公司/学校**：可编辑(必填)。

+ **简介**：可编辑(选填)。

依次填写可编辑项，点击**提交**按钮完成修改账户个人信息。

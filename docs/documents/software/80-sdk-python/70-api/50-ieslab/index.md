### `IESLabSimulation.createProjectGroup(group_name, desc=None, createById=None)`

- `desc`: [String][String] 项目组描述


### `IESLabSimulation.createProject(name, project_group, desc=None, createById=None)`

- 静态方法

## 安装

**CloudPSS** 可以在 [PyPi](https://pypi.org/) 上搜索直接下载，也可以通过 `pip` 安装更为便捷。只需运行以下命令即可安装：

```python
pip install cloudpss
```

## 升级
需要升级的话，执行下面的命令：

```python
pip install --upgrade cloudpss
```

## 卸载

执行下面的命令进行卸载：


```python
pip uninstall cloudpss
```

:::tip Python 版本
推荐使用 python 3.8 及以上版本。
:::

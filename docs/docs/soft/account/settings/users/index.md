> 此页面仅为管理员使用页面

点击页面左下角的账户`设置`按钮，然后点击左侧的`用户管理`按钮，进入用户管理页面。此页面以表格的形式列出网站所有注册的用户。表格内容包括：

+ 用户名：唯一标识

+ 邮箱

+ 电话

+ 公司：可缺省

+ 操作：修改密码

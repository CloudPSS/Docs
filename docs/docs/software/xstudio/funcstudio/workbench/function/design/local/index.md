## 常见问题

### 本地自定义命令可以直接在终端执行吗？

Funcstuio 为什么叫执行器，因为它是一个帮助用户在终端执行计算内核的工具，本质上相当于在本地打开命令窗口，先 cd 到计算内核所在的工作目录，再执行命令窗口里面的语句，因此上述命令也可以在终端执行。

:::tip

若本地只有一个 Python 环境，确认安装好 SDK 后，在命令窗口内直接输入 python adder.py 即可执行该函数。如下图所示：

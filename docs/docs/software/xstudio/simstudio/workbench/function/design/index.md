## 页面功能

用于进行当前项目的内核实现，可用于给普通项目构建仿真拓扑，也可用于元件项目的内核实现，针对元件项目实现方式有拓扑实现和代码实现，本节主要介绍拓扑实现，用户在该标签页内可进行元件的选取、拓扑的连接及构建、元件参数的配置、以及页面格式/元件格式的调整。关于代码实现详见XX文档。

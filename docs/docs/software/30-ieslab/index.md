IESLab 是 CloudPSS 团队基于 XStudio 基础支撑套件研发的一款面向**综合能源系统的数字孪生应用平台**，是拥有自主知识产权的国产化平台，按照使用目的不同分为**建模仿真**和**规划设计**两个子平台。能够灵活搭建综合能源系统拓扑，进行**仿真模拟、运行优化、设备选型定容、系统规划设计**，并开放 **SDK 开发套件**，让用户能够实现更多高级功能。如需深入了解各平台功能，请跳转相应帮助文档查阅。

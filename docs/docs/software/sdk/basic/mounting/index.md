## 安装

**CloudPSS** 可以直接从这里 [PyPi](https://pypi.org/) 下载，但是更方便的方式是通过 **pip**，直接运行下面的命令进行安装：

```python
pip install cloudpss
```

## 升级
需要升级的话，执行下面的命令：

```python
pip install --upgrade cloudpss
```

## 卸载

执行下面的命令进行完全卸载：


```python
pip uninstall cloudpss
```

:::tip
推荐使用 python 3.8 及以上版本。
:::

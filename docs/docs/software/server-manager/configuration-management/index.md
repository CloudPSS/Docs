**配置管理**页面是 **CloudPSS Server Manager** 的一个核心功能模块，它允许用户通过一个集中化的界面管理和配置 **CloudPSS** 软件及其服务。
这些服务的配置项和设置方法都以配置定义文件的形式进行描述。**Server Manager** 通过提供一个直观的图形界面，帮助用户轻松配置和存储这些配置。
